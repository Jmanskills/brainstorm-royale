## üèóÔ∏è Building System (Fortnite-Style!)

### Build Types:

**Wall**
- Vertical barrier
- Block incoming fire
- Quick cover

**Floor**
- Horizontal platform
- Create high ground
- Bridge gaps

**Ramp**
- Diagonal slope
- Run up for height advantage
- Quick escape

### Materials:

**Wood** (Brown)
- Health: 100
- Build Time: 1 second
- Fast but weak

**Brick** (Red)
- Health: 200
- Build Time: 1.5 seconds
- Balanced choice

**Metal** (Silver)
- Health: 300
- Build Time: 2 seconds
- Strongest but slowest

### How to Build:

1. **Press B** - Toggle build mode ON/OFF
2. **Press 1/2/3** - Switch build type (Wall/Floor/Ramp)
3. **Press Q/E** - Switch material (Wood/Brick/Metal)
4. **Click** - Place building at cursor
5. **Right-click** - Destroy your own building (get 5 materials back)

### Building Tips:
- Each placement costs 10 materials
- Start with 100 of each material
- Buildings can be destroyed by bullets
- Build time = vulnerable period before solid
- Bullets hit buildings before players (cover!)

### Strategy:
- **Defensive:** Build walls when taking fire
- **Offensive:** Build ramps to get height advantage
- **Mobility:** Build floors to cross dangerous areas
- **Creative:** Combine pieces for forts!
